```
    ↓

    ↓

    ↓

    ↓

```

## 快速上手指南

1. **打开 VS 后只看 3 个目录**：
   - `Server/Hotfix/` — 服务端业务逻辑
   - `Server/Model/` — 服务端数据定义
   - `Unity/Codes/HotfixView/` — 客户端界面逻辑

2. **找个已有的完整功能抄**：
   - 比如 `PlayerBag`（背包），从 Model → Hotfix → Proto → Handler → UI 完整抄一遍

3. **不要碰**（与核心框架无关）：
   - `Analyzer/`、`cwRsync/`、`RecastNavExportor/`、`ThirdParty/`
   - `Libs/Kcp/`、`Libs/RecastDll/`
   - `.vs/`、`Bin/`、`Logs/`、`Release/`

4. **从改已有功能开始**，而不是从零写
