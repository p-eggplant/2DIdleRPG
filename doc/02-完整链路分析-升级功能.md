# 完整链路分析：角色升级功能

本文以**角色升级**功能为例，带你从客户端按钮点击到服务端存盘，看清整个 ET 框架的数据流动。

---

## 功能全景图

```
客户端 Unity                               服务端 GameServer
  │                                              │
  ├─ 玩家点击升级按钮                             │
  │     │                                         │
  │     ▼                                         │
  ├─ 客户端校验（满级？材料？货币？）              │
  │     │                                         │
  │     ▼                                         │
  ├─ 发送 RPC 请求                                │
  │  Session.Call(C2Game_PlayerRoleUp) ──────────→ ├─ C2Game_PlayerRoleUpHandler
  │                                                 │   1. 验证参数合法性
  │                                                 │   2. LevelUp() → 改 m_CurLevel
  │                                                 │   3. RemoveItem() 扣材料
  │                                                 │   4. SubData() 扣货币
  │                                                 │   5. SetOne() 即时存盘
  │                                                 │   6. reply() 回复客户端
  │  ←─────────────────────────────────────────────┘
  │     │
  │     ▼
  ├─ 更新本地数据
  ├─ Tips("升级成功")
  └─ OnRefresh() 刷新 UI
```

## 逐层代码详解

### 第1层：UI 层（HotfixView）

**文件**：`Unity/Codes/HotfixView/Client/UIRoleUp/DlgRoleUpSystem.cs`

```csharp
// 用户点击升级按钮
public static async ETTask OnClickLevelUp(this DlgRoleUp self)
{
    // 1. 获取数据
    Player pPlayer = zoneScene.GetComponent<Player>();
    PlayerRoleUpComponent pRoleUp = pPlayer.GetComponent<PlayerRoleUpComponent>();
    PlayerBagComponent pBag = pPlayer.GetComponent<PlayerBagComponent>();
    PlayerDataComponent pData = pPlayer.GetComponent<PlayerDataComponent>();

    // 2. 客户端校验（减少无效网络请求）
    if (RoleLevelUpConfigCategory.Instance.IsMaxLevel(pRoleUp.m_CurLevel))
        { Tips("已达最大等级"); return; }
    if (pBag.GetItemNum(config.MaterialId) < config.MaterialNumb)
        { Tips("材料不足"); return; }
    if (pData.GetData(config.GetCurrencyEType()) < config.CurrencyNumb)
        { Tips("货币不足"); return; }

    // 3. 发 RPC 请求到服务端
    SessionComponent pSession = zoneScene.GetComponent<SessionComponent>();
    Game2C_PlayerRoleUp resp = (Game2C_PlayerRoleUp)await pSession.Session.Call(
        new C2Game_PlayerRoleUp());

    // 4. 处理服务端回复
    if (resp.Error != ErrorCode.ERR_Success)
        { Tips($"升级失败: {resp.Message}"); return; }

    // 5. 更新客户端数据 + 刷新 UI
    pRoleUp.m_CurLevel = resp.CurLevel;
    self.OnRefresh();
}
```

**关键点**：
- `self.DomainScene().GetComponent<Player>()` — 从场景获取玩家对象
- `Session.Call(request)` — RPC 调用，返回响应
- `await` — 异步等待，不会卡 UI

### 第2层：消息层（Proto）

**文件**：`proto/OuterMessage_PlayerRoleUp.proto` → 生成到 `Server/Model/Generate/Message/`

```csharp
// 请求消息（客户端 → 服务端）
[Message(OuterOpcode.C2Game_PlayerRoleUp)]
public partial class C2Game_PlayerRoleUp : Object, IActorRequest
{
    public int RpcId { get; set; }         // RPC 序列号
}

// 响应消息（服务端 → 客户端）
[Message(OuterOpcode.Game2C_PlayerRoleUp)]
public partial class Game2C_PlayerRoleUp : Object, IActorResponse
{
    public int RpcId { get; set; }
    public int Error { get; set; }          // 0=成功，非0=失败码
    public string Message { get; set; }     // 错误描述
    public int CurLevel { get; set; }       // 升级后的等级
}
```

**关键点**：
- `IActorRequest` / `IActorResponse` 标记请求和响应
- `Error = 0` 表示成功
- 消息由 `proto` 文件或手写定义，需在 `OuterOpcode` 注册 Opcode

### 第3层：服务端处理器层（Handler）

**文件**：`Server/Hotfix/Server/GameServer/PlayerSystem/PlayerRoleUpComponent/PlayerRoleUpComponentNetLogic.cs`

```csharp
public class C2Game_PlayerRoleUpHandler
    : AMActorRpcHandler<Player, C2Game_PlayerRoleUp, Game2C_PlayerRoleUp>
{
    protected override async ETTask Run(Player player, C2Game_PlayerRoleUp request,
        Game2C_PlayerRoleUp response, Action reply)
    {
        // 1. 获取玩家组件
        PlayerRoleUpComponent pRoleUp = player.GetComponent<PlayerRoleUpComponent>();
        PlayerBagComponent pBag = player.GetComponent<PlayerBagComponent>();
        PlayerDataComponent pData = player.GetComponent<PlayerDataComponent>();

        // 2. 各项验证
        if (pRoleUp == null) { response.Error = ...; reply(); return; }
        if (Already max level) { ... }
        if (!MaterialConfig exists) { ... }
        if (!Can remove items) { ... }
        if (!Can sub currency) { ... }
        if (!Can level up) { ... }

        // 3. 执行业务逻辑
        response.CurLevel = pRoleUp.LevelUp();           // 提升等级
        pBag.RemoveItem(...);                             // 扣除材料
        pData.SubData(...);                               // 扣除货币
        pRoleUp.SetOne();                                 // 即时存盘

        // 4. 回复客户端
        reply();
    }
}
```

**关键点**：
- 泛型参数：`<实体类型, 请求类型, 响应类型>`
- `reply()` **必须调用**，否则客户端会一直等待
- 所有验证失败都要设置 `response.Error`
